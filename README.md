# Anki AI标签添加器

这是一个使用AI为Anki卡片自动添加标签的工具，支持硅基流动等兼容OpenAI API格式的大模型。通过智能标签生成，帮助用户更好地组织和管理Anki卡片，提高学习效率。

## 功能特点

### 核心功能
- 🤖 **AI标签生成**：使用大模型智能分析卡片内容，自动生成相关标签
- 🔄 **牌组管理**：支持获取指定牌组的卡片，可选择处理所有牌组
- 📝 **灵活配置**：可自定义处理的卡片数量（5、10、20、50或全部）
- 🔍 **模型管理**：自动获取API支持的模型列表，支持模型搜索和筛选
- 🔧 **API配置**：支持多种AI服务（硅基流动、Gemini）和自定义API地址
- 📋 **Prompt定制**：支持自定义Prompt，控制标签生成风格，可保存多个预设

### 标签管理
- ✅ **标签校对**：批量检查卡片标签，对比现有标签与AI建议标签
- 🔎 **标签筛选**：支持按标签状态筛选（全部、有标签、无标签）和自定义字符筛选
- 🧹 **标签清理**：一键清理卡片的所有标签
- 🔄 **标签修正**：根据校对结果自动修正标签

### 用户体验
- 🎨 **图形界面**：直观的界面设计，支持滚动条和响应式布局
- ⚡ **多线程处理**：后台处理任务，避免UI卡顿
- ⏸️ **任务控制**：支持暂停和停止操作，更灵活的任务管理
- 👁️ **API Key安全**：支持API Key的显示/隐藏切换，保护敏感信息
- 💾 **配置保存**：自动保存API配置和Prompt设置，下次启动时自动加载

### 技术特性
- 🔌 **AnkiConnect集成**：通过AnkiConnect插件与Anki进行通信
- 🌐 **多模型支持**：兼容任何支持OpenAI API格式的大模型服务
- 📊 **实时预览**：预览标签生成结果，支持前5张卡片的快速预览
- 🛡️ **错误处理**：完善的错误捕获和提示机制，确保程序稳定运行
- 📈 **性能优化**：合理的API调用延迟，避免API限流

## 使用方法

### 1. 基本配置

1. **启动Anki**：确保Anki已打开，并且AnkiConnect插件已启用
2. **运行脚本**：
   ```bash
   python main.py
   ```
3. **配置AI参数**：
   - 输入API Key
   - 选择API服务（硅基流动、Gemini或自定义）
   - 点击"获取模型列表"按钮获取可用的模型
   - 选择要使用的模型

### 2. 牌组和卡片设置

1. **获取牌组列表**：点击"获取牌组列表"按钮
2. **选择牌组**：从下拉菜单中选择要处理的牌组（可选择"所有牌组"）
3. **设置卡片数量**：从下拉菜单中选择要处理的卡片数量（默认为"全部"）
4. **标签筛选**：选择标签筛选方式：
   - 全部卡片
   - 有标签的卡片
   - 无标签的卡片
   - 自定义筛选（可输入标签字符进行筛选）

### 3. Prompt设置

1. **选择预设Prompt**：从下拉菜单中选择预设的Prompt
2. **自定义Prompt**：在文本框中修改Prompt内容
3. **保存Prompt**：点击"保存当前Prompt"按钮保存自定义Prompt
4. **删除Prompt**：点击"删除当前Prompt"按钮删除选中的Prompt
5. **重置为默认**：点击"重置为默认"按钮恢复默认Prompt

### 4. 标签操作

1. **获取卡片**：点击"获取卡片"按钮，显示卡片列表
2. **预览标签**：点击"预览标签"按钮，预览AI生成的标签（仅前5张卡片）
3. **添加标签**：点击"添加标签"按钮，为所有卡片生成并添加标签
4. **清理标签**：点击"清理标签"按钮，移除所有卡片的标签
5. **校对标签**：点击"校对标签"按钮，批量检查卡片标签并显示建议
6. **修正标签**：点击"修正标签"按钮，根据校对结果修正标签

### 5. 任务控制

- **暂停**：点击"暂停"按钮暂停当前任务
- **停止**：点击"停止"按钮停止当前任务

## 注意事项

- 确保AnkiConnect服务正常运行（默认端口8765）
- API需要注册账号并获取API Key
- 操作过程中会有适当延迟，避免API限流
- 支持的模型列表会根据API响应自动更新
- 自定义Prompt时，请保持指令清晰，确保AI能够正确理解您的需求
- 校对标签功能会显示当前标签与建议标签的差异，方便您判断是否需要修正

## 兼容性

- 支持任何兼容OpenAI API格式的大模型服务
- 只需修改API Base地址和API Key即可切换到其他服务

## 故障排除

- **AnkiConnect连接失败**：检查Anki是否打开，AnkiConnect是否启用
- **API调用失败**：检查API Key是否正确，网络连接是否正常
- **模型列表获取失败**：检查API地址和Key是否正确
- **牌组列表获取失败**：检查Anki是否正常运行
- **标签生成失败**：检查API Key权限和网络连接
- **任务无响应**：尝试点击"停止"按钮，然后重新开始任务

## 技术架构

项目采用模块化设计，主要包含以下模块：

- **main.py**：主程序入口，处理用户界面和业务逻辑
- **modules/ai.py**：AI客户端，负责与大模型API交互
- **modules/anki.py**：AnkiConnect客户端，负责与Anki交互
- **modules/config.py**：配置管理，负责保存和加载配置
- **modules/ui.py**：UI管理，负责构建和管理用户界面
